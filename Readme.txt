packages/libraries needed to run this code:
numpy : pip install numpy
sklearn :  pip install -U scikit-learn
opencv : pip install opencv-python

HOW TO RUN:
open 'run.py' file
• change the variable TEST_DIRECTORY to the correct path of the test set (make sure it ends with /data/ folder)
	o	example:
  		TEST_DIRECTORY ='F:/cmp/fourth year/1st term/pattern recognition/project/sample/data/'
• change the variable test_sets_number to the correct number of test sets 
	o	example:
   		test_sets_number = 50
• save the file.
• run the file.
after the run, the results.txt and time.txt files will be next to the data folder
